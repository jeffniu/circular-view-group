# circular-view-group

Include following code to your module (app) build.gradle

```
allprojects {
    repositories {
        jcenter()
        maven { url "https://jitpack.io" }
    }
}
```

and include this line to dependencies

```
compile 'com.github.joginder89:circular-view-group:1.0.0'
```
